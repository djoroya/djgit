addpath_python(path) -> None¶

from addpath.addpath_python import addpath_python

Adds a custom path to Python's search path (sys.path) by creating or updating a .pth file inside the active environment's site-packages directory. This allows modules in non-standard locations to be imported without setting environment variables every session.

Section Description
Inputs --path (str): Absolute or relative path to add to the PYTHONPATH.
Outputs None (prints status messages to stdout and updates the conda.pth file).
  • The .pth file is created in the first site-packages found in sys.path.
  • If the path already exists in the file, it will not be duplicated.

Example usage:

djgit_addpath --path /path/to/my/path
